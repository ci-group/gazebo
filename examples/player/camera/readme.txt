= Player Camera Interface =

This example uses a mono cmaera  and the Player laser interface.

 1. Start Gazebo
  $ gzserver camera.world

 2. Start Player
  $ player camera.cfg

 3. Start the Gazebo GUI
  $ gzclient

 4. Start playerv
  $ playerv

 5. Subscribe to the camera interface using playerv. 

 6. In the Gazebo GUI add shapes to the world, and playerv will visualize the new camera images.
